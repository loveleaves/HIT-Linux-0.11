# Note



## 1.启动

主要是boot/目录中三个汇编代码文件：bootsect.s、head.s、setup.s。这三个文件虽然都是汇编程序，但却使用了两种语法格式。bootsect.s 和setup.s是实模式下运行的16位代码程序，采用近似于Intel的汇编语言语法并且需要使用Intel8086汇编编译器和连接器as86和ld86,而head.s则使用GNU的汇编程序格式，并且运行在保护模式下，需要用GNU的as (gas)进行编译。这是一种AT&T语法的汇编语言程序。

- 0x10号BIOS中断：显示字符串。
- 0x13号BIOS中断：读写磁盘的中断调用
- 获取硬件参数就是调用相应的BIPS 中断，显示信息的时候需要谨慎，因为
  返回的硬件参数是16 进制数字，需要转化成相应的ascii 码才能调用int 0x10 来
  输出。
